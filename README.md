# Travelgram
A travel profile page built using semantic HTML and CSS.

## 🛠️ Technologies Used
- HTML5
- CSS3

## 👀 How to View
1. Clone or download the repository.
2. Open `index.html` in your web browser.

## 🖼️ Assets
Images and icons are stored in the `assets/` folder.
